Statistics Module

mathtools.statistics.calculate_weighted_mean(values: List[float], weights: List[float] | None = None) → float[source]

Calculate the weighted arithmetic mean of a list of values.

Parameters:

  • values (List[float]) – List of numerical values

  • weights (List[float], optional) – List of weights corresponding to the values. If None, equal weights are assumed.

Returns:

The weighted arithmetic mean

Return type:

float

Examples

>>> values = [1, 2, 3, 4, 5]
>>> weights = [0.1, 0.2, 0.4, 0.2, 0.1]
>>> calculate_weighted_mean(values, weights)
3.0

Notes

The weighted mean is calculated as: Σ(value * weight) / Σ(weight)

mathtools.statistics.find_outliers(data: List[float], threshold: float = 2.0) → List[bool][source]

Detect outliers in a dataset using z-score method.

Parameters:

  • data (List[float]) – List of numerical values to check for outliers

  • threshold (float, optional) – Z-score threshold for outlier detection (default is 2.0)

Returns:

Boolean mask where True indicates an outlier

Return type:

List[bool]

Examples

>>> data = [1, 2, 100, 3, 4, 5]
>>> find_outliers(data)
[False, False, True, False, False, False]

Geometry Module

mathtools.geometry.calculate_circle_properties(radius: float) → Tuple[float, float][source]

Calculate the area and circumference of a circle.

Parameters:

radius (float) – The radius of the circle

Returns:

A tuple containing (area, circumference)

Return type:

Tuple[float, float]

Raises:

ValueError – If radius is negative

Examples

>>> area, circumference = calculate_circle_properties(1.0)
>>> print(f"Area: {area:.2f}, Circumference: {circumference:.2f}")
Area: 3.14, Circumference: 6.28
mathtools.geometry.compute_polygon_area(vertices: List[Tuple[float, float]]) → float[source]

Calculate the area of a polygon using the shoelace formula.

Parameters:

vertices (List[Tuple[float, float]]) – List of (x, y) coordinates of the polygon vertices in order

Returns:

Area of the polygon

Return type:

float

Notes

Uses the shoelace formula (also known as surveyor’s formula). Vertices should be ordered either clockwise or counterclockwise.

Examples

>>> # Calculate area of a triangle
>>> vertices = [(0, 0), (1, 0), (0, 1)]
>>> area = compute_polygon_area(vertices)
>>> print(f"Area: {area:.2f}")
Area: 0.50
